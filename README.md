# sql-mastery
SQLD 자격 취득 과정 및 데이터베이스 성능 최적화 쿼리 연구 기록 저장소

2027년 5월 31일 전역 이후 '아무것도 모르는 복학생'이 되지 않기 위한 이예성 일병의 몸부림

## 목표
  - 2026-03-07 SQLD 고득점 합격
  - sub query 및 window function 적용
  - 이론 공부에 그치지 않고 가상의 시나리오를 세워 쿼리를 작성한다
  - 성능저하가 예상되는 포인트에 대해 해결책을 기록한다

## 학습방법
  - '2026 시대에듀 유선배 SQL개발자(SQLD) 합격노트'의 예제 및 문제를 보고 어려웠던 문제를 분석한다
  - Local editor 를 사용할 수 없음으로 web sql 환경을 제공하는 [SQL Test](https://sqltest.net/) 에서 직접 쿼리를 작성한다.
